# Buggy Codebase Demo

Este repositório contém exemplos de código em diversas linguagens e formatos que apresentam falhas de lógica ou sintaxe. Ele foi criado para servir como alvo de testes para automações de correção de bugs (como o ResolveAI).

## Estrutura do Repositório

-   `python_bugs/`: Exemplos em Python.
-   `js_bugs/`: Exemplos em JavaScript (Node.js).
-   `ts_bugs/`: Exemplos em TypeScript.
-   `go_bugs/`: Exemplos em Go.
-   `configs/`: Arquivos de configuração (JSON, YAML).
-   `database/`: Scripts SQL.
-   `web/`: Páginas HTML.
-   `Dockerfile`: Configuração de container Docker.

## Objetivo

Use este repositório para avaliar o comportamento de ferramentas de diagnóstico de código e ferramentas de correção automatizada.
